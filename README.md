# 💜 Intima

> Интимен календар + дневник за теб и връзката ти. Криптиран, дискретен и само ваш.

## Структура на проекта

| Път                   | Описание                                                                             |
|-----------------------|--------------------------------------------------------------------------------------|
| `app/`                | Flutter приложението (Android + iOS)                                                 |
| `docs/BRAND.md`       | Име, цветова палитра, типография, тон                                                |
| `docs/COMPETITORS.md` | Конкурентен анализ и нашата ниша                                                     |
| `docs/MVP.md`         | Фиксиран MVP обхват + технологичен стек                                              |
| `IDEAS.md`            | Пълният списък с продуктови идеи                                                     |
| `PLAN.md`             | Фазов план + текущ статус (желязно правило: нова фаза само при 100% готова предишна) |

## Стартиране (development)

```powershell
cd app
flutter pub get
flutter run            # на стартиран Android емулатор/устройство
```

## Стек
Flutter · Riverpod · Drift + SQLCipher · go_router · local_auth

## Правило на проекта
⚠️ Никога не започваме нова фаза от `PLAN.md`, преди предишната да е 100% завършена (всички ✅ Definition of Done критерии — изпълнени и проверени).
