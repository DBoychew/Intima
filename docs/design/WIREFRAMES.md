## 3. Бърз запис „днес" (bottom sheet, < 10 сек)

```
┌──────────────────────────────┐
│        Как си днес?          │
│   😞   😐   🙂   😊   🥰      │   ← избор на настроение
│                              │
│ ( Интимен момент ♥ )         │   ← при избор се РАЗГЪВА ↓
│ ( Менструация ) ( ПМС )      │
│ ( Главоболие ) ( Лош сън )   │
│ ┌─ ♥ Момент 1 ──────────  ✕ ┐│
│ │ Възбуда  ────────▮────     ││
│ │ Оргазми   [−]  1  [+]      ││
│ │ Пози: (Мисионерска)        ││
│ │ (Лъжички)(Отгоре)(Отзад)   ││
│ │ (Права)(69)(Друга)         ││
│ │ ┌────────────────────────┐ ││
│ │ │ Опиши момента…         │ ││   ← описание, само за теб
│ │ └────────────────────────┘ ││
│ └────────────────────────────┘│
│   + Добави още един момент   │   ← няколко момента на ден
│                              │
│ Либидо   ────────▮────       │
│ Енергия  ──────▮──────       │
│  ┌────────────────────────┐  │
│  │       Запази ✨        │  │
│  └────────────────────────┘  │
└──────────────────────────────┘
```

> Във Фаза 8 списъкът с пози се замества от библиотеката с илюстрации —
> тук позите са текстови chips, за да валидираме интеракцията.
