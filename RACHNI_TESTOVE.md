   памет при снимки, смяна на часова зона (напомняния)

2. Кабел към компютъра → потвърди отпечатъка на компютъра
3. `cd C:\Users\dimitar.k.boychev\Desktop\Diary\app`
4. Debug: `flutter install -d <device-id>` /
